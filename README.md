# 🍪 Cookie Clicker in Minecraft 
This is a recreation of Cookie Clicker in Minecraft

## ⭐ Current features:
- Buildings up to the wizard tower
- Upgrades for buildings
- Cps upgrades
- Milk system
- Kitten upgrades
- Game saving

Controls:
- Press 'c' to buy buildings
- Press 'v' to buy upgrades
- Press ',' to toggle game
